## 1. Objective

Ensure all investors and high-value actors undergo:

* Identity verification
* Anti-money laundering checks
* Jurisdiction compliance screening

---

## 2. KYC Lifecycle

States:

* PENDING
* DOCUMENT_SUBMITTED
* UNDER_REVIEW
* VERIFIED
* REJECTED

---

## 3. Required Verification Inputs

Investors must provide:

* Government-issued ID
* Proof of address
* Source of funds declaration

---

## 4. AML Screening

Checks include:

* Sanctions list screening
* Politically exposed person (PEP) flagging
* Transaction monitoring alerts

---

## 5. Platform Enforcement

* Only VERIFIED investors can fund projects
* Suspicious accounts trigger holds
* Admin compliance overrides required

---

## 6. Phase 6+ Upgrade

Future enhancements:

* Automated third-party KYC providers
* Continuous monitoring
* Regulatory reporting APIs
